A fully responsive, production-grade Netflix Clone built with HTML5, CSS3, and JavaScript. This application integrates with the TMDB API to fetch real-time movie data, trending content, and video trailers.

** Live Demo:** [Click Here to Watch](https://serene-biscotti-5e397b.netlify.app/)

# Features
- **Authentication Flow:** Custom Login, Sign Up, and Help pages with form validation.
- **Dynamic Content:** Fetches "Trending Now", "Netflix Originals", and "Top Rated" movies dynamically using the TMDB API.
- **Video Playback:** Integrated YouTube trailer playback system using the `movie-trailer` library.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop (CSS Grid & Flexbox).
- **User Interface:** Pixel-perfect replication of the Netflix landing page and browse experience.

# Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **API:** The Movie Database (TMDB)
- **Deployment:** Netlify
- **Version Control:** Git & GitHub
